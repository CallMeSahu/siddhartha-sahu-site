Getting Started with Programming


So you finally decided to try out computer programming. The next hurdle you will face is to get started. Don't worry, this blog is exactly what you need. I want you to choose any language of your choice, I suggest choose and follow this checklist along. My programming language suggestions would be C++/Java. These languages are been here for a while and you will find plenty of resources for studying each topic. Now for a particular topic use GeeksForGeeks, Youtube videos or any textbook. After learning the particular topic just go to google and find questions mentioned on GeeksForGeeks or LeetCode. Read the question well and give yourself at least 30 minutes to come up with a solution and then code it out. If you are unable to come up with a solution use hints and editorial provided to you and try to code again without seeing the solution. It is important for you to not copy the solution and train yourself to come up with the logic. This roadmap also serves as a prerequisite to Competitive Programming and Advanced Data Structures and Algorithm problems asked in tech interviews. I suggest you invest at least 200 hours to complete this program before moving to something else. With that said LET'S GET STARTED!

  1. Basics Concepts
    • Getting Started: Print function, Variables, Conditional statements, Loops, Operators, prime numbers, fibonacci series, digits in a number, reverse a number, rotate a number, GCD & LCM, Prime factors.
    • Patterns: Print Z, Square, Rectangle, Triangle, Inverted Triangle, Pyramid.
    • Functions and Arrays: Functions concept, Array concept, Span array, Find element in an array, Reverse an array, Rotate an Array, Sum of two arrays, Difference of two arrays, First index last index, Subsets of an array.
    • 2-D Arrays: 2-D array concept, Matrix multiplication, Spiral display, End-point of matrix, Rotate by 90-degree, Search in 2-D matrix.
    • Strings and ArrayList: String concept, Stringbuilder function, ArrayList concept, Pallindromic substring, String compression, Toggle case, Print all permutations, Remove primes.
  2. Recursion and Backtracking
    • Introduction to Recursion Print increasing-decresing, factorial, Power linear, Power logarithmic, Print zigzag, Tower of hanoi
    • Recursion in Array Display array, Display in reverse, Max of array, First index, Last index, All indices in array
    • Recursion of ArrayList Get subsequence, Get Kpc, Stair Path, Maze Path.
    • Rucursion on the way upSubsequence, KPC, Stair path, Maze path, Permutation, Encoding.
    • Recurssion BacktrackingFlood-fill, Target sum subsets, N queens, Knight tour
  3. Dynamic Programming
    • DP and Greedy Fibonacci, Climb stairs, Min cost in maze, Target sum subsets, Coin change, Knapsack, Count Binary String, Arrange Building, Count Encoding, Paint fence, Tiling with M tiles, Friends Pairing, Partition into subsets, Buy and sell stocks.
    • Time-Complexity Asysmptotic notation concept, Time complexity concept, Bubble-sort, Selection-sort, Insertion-sort, Merge-sort, Quick-sort, Count-sort, Radix-sort, Target sum
  4. Object-Oriented Programming
    • OOPs Concepts Objects, Class, Pointers, Inheritance, Polymorphism, Abstraction, Encapsulation.
  5. Basics Data Structures
    • Stacks and Queues Stack concept, Balanced brackets, Next greater element, Stock span, Largest area histogram, Sliding window, Infix conversion, Merge Overlapping interval, Dynamic stack, Minimum Stack, Queues concept, Dynamic Queues, Queue Stack Adapter, Two stack in an array.
    • Linked List Add last in LL, delete first in LL, Add at index LL, remove last in LL, reverse LL, LL to stack adapter and queue adapter, Kth node from end in LL, Merge sort in LL, Remove duplicates in LL, Odd even in LL, K reverse in LL, Palindrome LL, Fold a LL, Intersection point in LL.
    • Generic Tree Generic tree concept, Size of GT, Level order linewise-zigzag, Mirror a GT, Find in GT, Same shape GT, Mirror GT, Kth largest elemnt in tree, node with maximum subtree sum, diameter in GT,
    • Binary Tree Binary tree concepts, size sum heigth of BT, Traversals in BT, Node to root path, Print k level down, Transform to left clone, Remove leaves in BT, Tilt BT, Largest BST subtree
    • Binary Search Tree BST Concept, Size Sum Max Min in BST, Add and remove node, LCA of BST, Target sum pair.
  6. Advanced Data Structures
    • Hashmaps and Heaps Hashmaps concept, Highest Frequency character, Get common element, Longest consecutive sequence, Heap concept, K largest element, Sort k sorted array, Median priority queue, Merge k sorted list.
    • Graphs Graph concept, Has path, Print all path, Get connected components, Number of islands, Breadth first traversal, Is graph cyclic or bipartite, Order of compilation
